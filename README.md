# Peluqueria Landing Demo

Demo estatica para presentar una landing page orientada a peluquerias y barberias.

## Abrir

Abre `index.html` en el navegador.

## Archivos

- `index.html`: estructura de la pagina.
- `styles.css`: estilos responsivos.
- `script.js`: interacciones simples de demo.
- `assets/salon-hero.png`: imagen hero generada para el proyecto.
